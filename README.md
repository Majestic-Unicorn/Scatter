# Controls
Gamepads are required as keyboard is no longer supported.  
  
**Xbox 360 Controls:**  
Left Joystick = Movement  
A = Attack  
B = Lay Trap  
Y = Sprint  

# Requirements
Download UnityVS for your version of Visual Studio to open the solution:  
https://www.visualstudio.com/features/unitytools-vs
