# Controls
Gamepad is required as keyboard is no longer supported.  
  
**Xbox 360 Controls:**  
Left Joystick = Movement  
A = Attack  
B = Lay Trap  
RB = Sprint  

# Requirements
Download UnityVS for your version of Visual Studio to open the solution - http://unityvs.com/
